Welcome !

This is an app to store Bookmarks based upon the Type of resource the bookmark correlates to.

To add a Bookmark, first create the type by navigating to the add types section using the navigation bar at the top of the page.

Once the type is created, navigate to the add bookmarks page to add your bookmarks to your newly created type.
